### **FRAMEWORK 1: Quick Magnet v2.8.3 (Thai Social Media)**

**Specifications:**
- Length: 160-200 words | Reading Time: 40-50 seconds
- Thai-First Handshake: 85-95% (MANDATORY)
- Frequency: Mon-Fri (5x/week)

**4-Part Structure:**

```markdown
[HOOK - Choose Variant A or Variant B]

**Hook Variant A (Standard):** The Tension Hook (Problem → Agitate)
**Hook Variant B (The Belief Flag):** The Sinek Filter (Belief → Alienate)

*Structure for Variant B:*
"ผมเชื่อว่า [Controversial Truth e.g., ประกันคือโล่ ไม่ใช่หวย].
ถ้าคุณมองหา [Mainstream Desire e.g., 'ผลตอบแทนเร็วๆ'], หยุดอ่านตรงนี้ได้เลยครับ
แต่ถ้าคุณให้ค่ากับ [Core Value e.g., 'ความแน่นอน'], โพสต์นี้เขียนมาเพื่อคุณ"

*Goal:* Filter out the 68% Majority immediately; Lock in the 15% Early Adopters.

⚠️ **VARIETY RULE (Must Rotate):**
Do NOT always start with "ผมเชื่อว่า...". Use these variations:
1. **The Refusal:** "เราไม่เชื่อเรื่อง [Popular Action]." (e.g., "เราไม่เชื่อเรื่องการเก็งกำไรในประกัน")
2. **The Definition:** "[Concept] จริงๆ แล้วคือ [Reframed Definition]." (e.g., "ความมั่งคั่งไม่ใช่ตัวเงิน...")
3. **The Filter:** "ถ้าคุณหา [X] โพสต์นี้ไม่ใช่สำหรับคุณ" (e.g., "ถ้าคุณหาทางลัด...")

Expose the misconception

### [Emoji] The Machine (3-4 sentences OR 2-3 bullets)
Explain mechanism - MANDATORY section
Show A → B → C causality

### [Emoji] The Leverage (2-3 sentences)
Smart asymmetric move

### [Emoji] The Verdict (สรุปสั้นๆ) (1-2 sentences)
Sharp instruction or equation (A = B)

👉 [Link]
```

**Emoji Pools:**
| Section | Options |
|---------|---------|
| Reality Check | 🏛️ 🧠 📉 ♟️ 🎭 |
| Machine | ⚙️ 🗝️ 🧩 🔍 ⚗️ |
| Leverage | 💡 🗝️ 🛡️ 📈 🎯 |

- Thai-first ratio MUST be 85%+

### **FRAMEWORK 4: Formal Dispute Letter (Official)**

**Use Case:** Official disputes, medical/insurance appeals, government correspondence

**6-Part Structure:**

```markdown
[HEADER]
วันที่: [Date] | เรื่อง: [Subject] | เรียน: [Addressee]

[1. SUBJECT/CONTEXT] - 1-2 paragraphs
State issue, reference policy numbers, dates, prior communications

[2. IDENTIFY THE FLAW] - 1 paragraph
"อาจเกิดจากความเข้าใจคลาดเคลื่อน"

[3. PRESENT EVIDENCE] - Numbered list (5-10 points)
Chronological: dates, expert names, documents, amounts

[4. PROVE THE LOGIC] - 2-3 paragraphs
Connect evidence to conclusion

[5. PROVIDE SOLUTION] - 1-2 paragraphs
Specific request, offer cooperation

[6. PROFESSIONAL CLOSE]
"จึงเรียนมาเพื่อโปรดพิจารณา"

[SIGNATURE]
นาย ด. และ น.ส. ส.
```

**Language Rules:**
- Thai: "ข้าพเจ้า" ALWAYS (ultra-formal), pure Thai, children = initials only
- English: Minimize first-person, professional register, no contractions

- Mon-Fri: 5 Quick Magnets
- Sat-Sun: 2 Kane Editions

**Process (3-4 hours):**

| Phase | Time | Task |
|-------|------|------|
| Preparation | 30 min | Review last week, list 7 angles, assign frameworks |

| Batch 2 | 60-90 min | Both Kane Editions (deeper mode) |

**Rules:**
- ✅ Batch similar frameworks together
- ✅ Verify each piece immediately after writing
- ❌ Don't mix Quick and Kane in same session
- ❌ Don't skip verification "to save time"

<a name="quality-gates"></a>
## 🔐 **QUALITY GATES (5 Mandatory Checks)**

Every piece MUST pass ALL 5 gates before publishing.

### **GATE 1: Brand Constitution Compliance**

```
□ No status terms (HNW, Rich, Wealthy, รวย, มั่งคั่ง)
□ No urgency manipulation (Hurry, โปรโมชั่น, อย่าพลาด)
□ Privacy protected (children = initials ALWAYS)
□ Evidence-based claims (specific data, not vague)
□ Mechanism explained (The Machine section present)
```

### **GATE 2: Voice DNA Alignment**

```
□ Sounds authentically like you (read aloud test)
□ Correct pronoun:
  - ข้าพเจ้า for formal Thai letters
  - ผม (minimal) for Thai social
  - You-focused for English
□ Thai-First Handshake verified (if Thai social)
□ Tone calibrated for platform
```

### **GATE 3: Terminology Accuracy**

```
□ Critical terms correct:
  - มูลค่าเวนคืน (NOT จำนำ)
  - การเลือกที่ขัดกัน (NOT ผิด)
  - ความไม่สมมาตร (NOT ไม่เท่าเทียม)
  - มูลค่าเงินสด (NOT เงินสะสม)
□ Fiduciary terms context-appropriate
□ No LaTeX symbols ($$) in social content
```

### **GATE 4: Structural Integrity**

```
□ Correct framework used
□ Word count in range (Quick: 160-200, Kane: 300-350, Legacy: 300-400)
□ All required sections present
□ Headers formatted correctly
□ No meta-commentary ("In this post, I will...")
```

### **GATE 5: Final Polish**

```
□ Micro-paragraphs (2-3 sentences max)
□ White space adequate (mobile readability)
□ Visual hierarchy clear
□ CTA present (👉 for social)
□ Ready to publish AS-IS (0 edits needed)
```

**If ALL checked → PUBLISH | If ANY unchecked → REVISE**
