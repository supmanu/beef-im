## 📝 CITATION FORMAT EXAMPLES

### **Footnote Style (Thai Social Media):**

**In Text:**
```
กองทุนประกันสังคมมีสำรอง 2.7 ล้านล้านบาทในไตรมาสแรกของปี 2025[⁴] 
และคาดว่าจะหมดในปี 2054 หากไม่มีการปฏิรูป[⁵]
```

**Reference Section:**
```
[⁴] SSO (2025), "Thailand's Social Security Fund Q1 2025," The Nation, May 2025
[⁵] OECD (2025), "Financing Social Protection in Thailand," Official Report
```

### **Inline Style (Academic/Blog):**
```
According to the Social Security Office (SSO, 2025), the fund held 
reserves of 2.7 trillion baht. Research by the OECD (2025) suggests 
these reserves will be depleted by 2054 without reform.
```

### **Hybrid Style (Whitepapers):**
```
The Social Security Fund held reserves of 2.7 trillion baht[¹] but faces
depletion by 2054 without reform[²].

[1] SSO (2025). "Q1 2025 Fund Performance." URL: [link]
[2] OECD (2025). "Financing Social Protection in Thailand." URL: [link]
```

## 🚨 COMMON MISTAKES

| Mistake | Fix |
|---------|-----|
| **Over-Citing** | Only cite quantitative claims, projections, policies |
| **Under-Citing** | Back critical facts with authoritative sources |
| **Weak Sources** | Use official sources for official data |
| **Broken Citations** | Verify every citation before publishing |
| **Format Inconsistency** | Pick one style and use consistently |

## 💡 PRO TIPS

1. **Batch Searches:** Do all source searches in one 30-45 min session
2. **Bookmark Key Sources:** Create browser folder for Thai official sources
3. **Date Matters:** Prioritize sources from last 12 months for dynamic topics
4. **Screenshot Critical Data:** Websites change; save key tables/charts
5. **Cross-Verify:** For critical facts, find 2-3 independent sources
6. **Read Actual Source:** Don't cite based on headlines; verify claim accuracy
