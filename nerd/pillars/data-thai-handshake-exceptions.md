# Thai-First Handshake: Exception Categories
## v4.3.1 OPTIMIZED

**Purpose:** Clarifies exceptions to the 85/15 Thai-First rule  
**Updated:** November 27, 2025 (Optimized Edition)

---

## CORE PRINCIPLE (UNCHANGED)

**Thai-First Handshake: 85-95% for ALL explainable concepts.**

**Standard Pattern:**
```
✅ ความเสี่ยงภัย (Risk)
✅ ต้นทุนทบต้น (Compounding Cost)
✅ มูลค่าเงินสด (Cash Value)
```

---

## EXCEPTION CATEGORIES

These are **EXEMPT from the 85/15 count** — they do NOT count as bilingual terms.

---

### **CATEGORY 1: PROPER NOUNS (People)**

**Rule:** Use English name, optional Thai descriptor (not translation)

| ✅ CORRECT | ❌ WRONG |
|-----------|---------|
| Ray Dalio | ราย ดาลิโอ (Ray Dalio) |
| Naval Ravikant | นาวัล ราวิกานท์ (Naval Ravikant) |
| Warren Buffett | วอร์เรน บัฟเฟตต์ |

**With descriptor OK:** `Ray Dalio นักลงทุนระดับตำนาน`

**Why:** International recognition preserved; Thai transliterations awkward

---

### **CATEGORY 2: EPONYMOUS TERMS**

**Rule:** Brief Thai explanation + English term in parentheses

| ✅ CORRECT |
|-----------|
| หลัก Pareto (Pareto Principle) |
| สมดุลแนช (Nash Equilibrium) |

---

### **CATEGORY 2A: CONTROVERSIAL FINANCIAL TERMS (English-Only)**

**Rule:** Use English term ONLY — Thai translations carry legal/criminal connotations

| Term | Usage Pattern |
|------|---------------|
| Ponzi Scheme | "มันไม่ดึงความมั่งคั่งออกมาครั้งเดียว (Ponzi Scheme)" |
| Pyramid Scheme | "ระบบนี้ไม่ใช่ (Pyramid Scheme)" |
| Pump and Dump | "เทคนิค (Pump and Dump) ถูกห้ามโดย SEC" |
| Front-running | "พฤติกรรมนี้เป็น (Front-running)" |
| Insider Trading | "พฤติกรรม (Insider Trading) ถูกห้ามโดย SEC" |

**Exception:** Formal legal documents citing specific laws may use Thai legal terminology

---

### **CATEGORY 3: REGULATORY ACRONYMS**

**Rule:** Thai full name on first mention + acronym, then acronym only

**First Mention:**
```
สำนักงานคณะกรรมการกำกับหลักทรัพย์และตลาดหลักทรัพย์ (SEC)
สำนักงานคณะกรรมการกำกับและส่งเสริมการประกอบธุรกิจประกันภัย (OIC)
องค์การอาหารและยา (FDA)
```

**Subsequent:** `SEC กำหนดให้...`, `OIC เปิดเผยว่า...`

**Common Acronyms (Exempt):**
SEC, OIC, BOT, FDA, IMF, WHO, IFRS, GAAP

---

### **CATEGORY 4: GLOBAL BRAND NAMES**

**Rule:** English brand name + optional Thai descriptor

| ✅ CORRECT | ❌ WRONG |
|-----------|---------|
| Apple บริษัทเทคโนโลยียักษ์ใหญ่ | แอปเปิล (Apple) |
| Tesla ผู้ผลิตรถยนต์ไฟฟ้า | เทสลา (Tesla) |
| iPhone สมาร์ทโฟนรุ่นยอดนิยม | ไอโฟน |

**Generic vs Brand:**
- Generic: "สมาร์ทโฟน (Smartphone)" — Thai-first OK
- Brand: "iPhone ของ Apple" — English brand name

---

### **CATEGORY 5: TECHNICAL STANDARDS**

**Rule:** Use acronym/code as-is (international standards)

```
✅ ISO 9001 (มาตรฐานการจัดการคุณภาพ)
✅ IFRS (มาตรฐานการรายงานทางการเงินระหว่างประเทศ)
✅ GAAP (หลักการบัญชีที่รับรองทั่วไป)
✅ Basel III (ข้อตกลงบาเซิล 3)
```

---

### **CATEGORY 6: SIGNATURE VOICE TERMS (Your Brand)**

**Rule:** These are YOUR signature concepts — English-first by design

| Term | Usage |
|------|-------|
| Leverage | "Leverage คือ..." |
| Audit | "ทำ Audit กรมธรรม์" |
| Arbitrage | "โอกาส Arbitrage" |
| Simple Swap | "ใช้ Simple Swap framework" |

**These are INTENTIONALLY English-first** for brand voice positioning.

## COUNTING METHODOLOGY

### **How to Calculate Thai-First Ratio:**

```
STEP 1: Count ALL bilingual terms
STEP 2: Subtract exception categories (1-6)
STEP 3: Count Thai-first among remaining
STEP 4: Calculate: Thai-first ÷ Remaining = Ratio

EXAMPLE:
Total bilingual terms: 40
Exceptions: 7 (Ray Dalio, Ponzi, SEC, OIC, Apple, Leverage, Audit)
Countable terms: 40 - 7 = 33
Thai-first count: 31
Ratio: 31 ÷ 33 = 93.9% ✅ PASS
```

## EDGE CASES

### **When Unsure:**

| Question | Answer | Action |
|----------|--------|--------|
| Can concept be naturally explained in Thai first? | YES | Thai-first pattern |
| | NO | Check Categories 1-6 |
| Still unsure? | — | Default to Thai-first (safer) |

### **Compound Technical Terms:**

```
"Cash Flow Arbitrage"
→ Treat as single concept
→ "โอกาสจากความไม่สมมาตรของกระแสเงินสด (Cash Flow Arbitrage)"
```

### **Industry Jargon vs Proper Terms:**

| Term | Treatment |
|------|-----------|
| Alpha (investment) | Thai-first: "ผลตอบแทนส่วนเกิน (Alpha)" |
| Beta (risk) | Thai-first: "ความเสี่ยงเชิงระบบ (Beta)" |

**Why:** Alpha/Beta are explainable concepts, not proper nouns

## FINAL REMINDERS

| ✅ DO | ❌ DON'T |
|-------|---------|
| Use exceptions sparingly | Abuse exceptions to avoid Thai |
| Count and verify ratio | Skip verification |
| Default to Thai-first when unsure | Use exceptions as excuse for lazy English |

**Philosophy:** Thai-First Handshake makes sophisticated concepts accessible. Exceptions exist for terms MORE accessible in English (proper nouns, brands) or internationally standardized (acronyms).

**When in doubt:** "Which language makes this MORE clear to my Thai reader?"
