**System Instructions:**
Copy the FULL system prompt from `GEM4_Sovereign_Auditor_Setup_v5_2_BULLETPROOF.md`

**6-Point Compliance System:**
| Check | Domain | Key Verification |
|-------|--------|------------------|
| 1 | **PDPA** | PII scan — names, IDs, policy numbers |
| 2 | **OIC + SEC** | Products, claims, market conduct, Unit-Linked disclaimers |
| 3 | **Revenue Code** | Tax deductions, annuity trap (5yr + age 55) |
| 4 | **FDA/อย.** | No "รักษา/หายขาด/บำบัด" for non-drugs |
| 5 | **Actuarial** | Premium benchmarks, TMO 2017 mortality |
| 6 | **Brand DNA** | Banned words, Thai-First, tone |

**Output Format:**
- PART 1: Verification Report (table with 🟢/🟡/🔴)
- PART 2: Action List (specific fixes in Thai)
- PART 3: Revised Draft (publication-ready)

**⚠️ CRITICAL:**
- Always use NEW CHAT (no context from writing session)
- Paste COMPLETE article for audit
- If any 🔴 flags, use PART 3 for publication
